🧠 MindMap Interactive

An interactive Mind Mapping web application built using HTML, CSS, and JavaScript that allows users to visually organize ideas using draggable nodes on a canvas.

This project helps users brainstorm, structure thoughts, and create visual idea maps.

✨ Features

🧠 Central Root Idea Node

➕ Add new idea nodes

🖱 Drag and drop nodes anywhere

🔗 Lines connecting nodes

🎨 Color categories for ideas

🗑 Delete node option

💾 Auto save using LocalStorage

🔍 Zoom in / Zoom out canvas

🌄 Modern UI with background image and glassmorphism

🛠 Technologies Used

HTML5 – Structure of the application

CSS3 – Styling and UI design

JavaScript (Vanilla JS) – Application logic and interactivity

SVG – Drawing connection lines between nodes

LocalStorage – Saving user data
